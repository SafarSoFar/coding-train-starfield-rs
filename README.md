# The Coding Train Starfield implementation in Rust & Raylib
The implementation of https://www.youtube.com/watch?v=17WoOqgXsRM from original **JavaScript & p5.js** to **Rust & Raylib**

```

cargo build
```

*from crates.io*<br>
